Automatic Number plate recognition - We built a yolov4-tiny model for license plate detection. - For recognition of characters on the license plate , we used paddle OCR (we built on edge device raspberry pi 4) 

Facial Recognition - We built facial recognition API , with the help of the API we can add , delete the users and recognize the users. 

Try on clothes - We built a virtual try on clothes system which will take a model image and the apparel image , and will produce an output which will show the model wearing that apparel. 

Cancelable Biometrics using Deep Learning - Used Star and Bubble Ordinal Patterns based LBP-CNN to extract features and transformed using Bio-Hashing into a non-invertible and revocable template. 

Speech to Text - We built tensorflow implementation of speech recognition based on DeepMind's WaveNet: A Generative Model for Raw Audio.  

Human Action Recognition - Human action recognition in video is of interest for applications such as automated surveillance, elderly behavior monitoring, human-computer interaction, content-based video retrieval, and video summarization. In monitoring the activities of daily living. We used vision Transformers for Human Action Recognition . 

Secure Encrypted Template Generation for Iris ROI Strip - Building a CNN model with non-trainable ordinal Gaussian filters. · Encrypted Iris ROI features using Encryption scheme based on Hyperchaotic Rossler Map. 

portrait segmentation of image - Portrait segmentation refers to the process of segmenting a person in an image from its background. - Here we use the concept of semantic segmentation to predict the label of every pixel (dense prediction) in an image. 

semantic search engine on stackoverflow python data - using universal sentence encoder of stackoverflow python questions titles and based upon similarities of universal sentence enoder vectors ,predict similar python questions of stackoverflow.  

Image Captioning Using Encoder-Decoder and Flickr Dataset - Automatically generating the textual description from an artificial system is the task of image captioning. - Describing the activity in the Image. 

Chatbot for Restaurant - The chatbot is designed using an open source machine learning framework Rasa. The extracted entities are parsed according to defined user intents and the values are stored in appropriate slots. The form actiavtes a loop which keeps on asking the user for the required entities. The bot is also capable of handling chitchats and answering frequently  

Automatic Information Extraction from Documents - We used Layout LMV3 model for automatic information extraction from pdfs and we collected the data from different resources and annotated using label studio.
